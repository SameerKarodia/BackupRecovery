**How To Run:**   
1. Open up Linux.

2. Open Terminal.

3. Type in “ pwd ” to see current directory.

4. Navigate to the home directory [cd  /home/$USER ].

5. Create a backup.sh file and paste code in the file.  
[ touch backup.sh -> vim backup.sh , paste code into script ]  
OR   
download the backup.sh file and move it to the directory [  /home/$USER ]

6. Give permissions to backup.sh to execute commands [ chmod 777 backup.sh ].

7. Create a backups directory to store the backups [ mkdir backups ].
